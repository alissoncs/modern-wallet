# Running the project
Recommended NODE.js Version: v13.11.0

In the root of repository, run
```
$ npm install
```
or 
```
$ yarn
```

## Run the Rest API
Go to the `/api` folder, run
```
$ nodemon index.ts
# # or
$ npm run start
# or 
$ yarn run start
```
## Run the webapp
Go to the `/webapp` folder, run
```
$ npm run start
```

That's all!
